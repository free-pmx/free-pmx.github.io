free-pmx: Insights on Proxmox

Maintained content at `free-pmx.pages.dev <https://free-pmx.pages.dev>`__. Current `list of posts <https://github.com/free-pmx/free-pmx/blob/main/LIST.rst>`__ and `RSS & ATOM feeds <https://free-pmx.pages.dev/feed/>`__ available.
